Definition List – Web Technologies

HTML
HyperText Markup Language – used to structure web pages.
CSS
Cascading Style Sheets – used to style and format web content.
JavaScript
A scripting language that adds interactivity to web pages.
Frontend
The part of a website that users see and interact with directly.
Backend
The server-side component that handles data, logic, and communication with databases.
